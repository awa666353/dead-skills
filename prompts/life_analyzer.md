# 生平记忆分析 Prompt（逝者）

## 任务

你将收到关于 **{name}** 的原材料（聊天记录节选、悼文、日记、子女回忆、照片说明等）。
从中提取可核验的 **生平与记忆**，用于构建 `life.md`（PART A）。

**原则**：
- 只写与 **{name}** 强相关、有材料支撑的内容；无依据则写「原材料不足」。
- 使用 **第三人称** 或 **中立追述**（「他/她/TA」），不要写成「我」的自传，除非引号内为逝者原话。
- 不做临床心理诊断；不臆测隐私。

---

## 提取维度

### 1. 价值观与看重的事

- 对工作、家庭、金钱、信仰（如有提及）、「面子/情分」等的态度
- 人生晚期或关键阶段明显变化（如有材料）

```
输出格式：
工作/事业：[描述或 原材料不足]
家庭观：[描述]
金钱观：[描述]
其他坚守：[描述]
```

### 2. 生活习惯与日常

- 作息、饮食、爱好、常去的地方、固定仪式（喝茶、散步、看球等）

```
输出格式：
作息：[描述]
饮食/嗜好：[描述]
空间/去处：[描述]
日常仪式：[描述]
```

### 3. 重要人生节点

- 升学、婚育、迁徙、职业转折、疾病与康复（如公开记述）、离世前后亲友集中提及的事

```
输出格式：
时间线要点：[按时间列出，不确定的年份标「约」]
反复被提及的场景：[列表]
转折点：[事件 + 对周边人的影响（如有）]
```

### 4. 人际关系（以 TA 为中心）

- 对配偶、子女、手足、老友、邻里的典型态度与互动方式
- 冲突处理风格（若材料有）

```
输出格式：
对家庭：[描述 + 典型场景（若有）]
对亲友：[描述]
对外界/生人：[描述]
```

### 5. 精神遗产与他人眼中的 TA

- 后辈、同事复述的教诲、笑话、习惯
- 「大家都记得 TA 这一点」式概括（需有样本，不可空泛）

```
输出格式：
常被转述的话：[列表]
集体记忆点：[需有至少一则材料支撑]
```

### 6. 晚期变化与代际关系（如有材料）

- 退休、疾病、丧偶等之后 **可见的行为或心态变化**（仅转述，不诊断）
- 与孙辈、子女相处方式的演变
- 对「被照顾」的态度（若材料有）

```
输出格式：
晚期变化：[描述或 原材料不足]
与晚辈：[描述或 原材料不足]
```

---

## 输出要求

- 语言：中文
- 缺信息维度：标注 `（原材料不足）`
- 有依据的结论：可附短引号原话
- 避免将活人「愿望」强加为逝者「遗言」；区分 **转述** 与 **推断**
