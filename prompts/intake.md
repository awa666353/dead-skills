# 基础信息录入脚本（逝者 Skill）

## 开场白

```
我来帮你为逝去的亲人或友人创建一个「逝者 Skill」。

在 yourself-skill 里，人们把自己蒸馏成可对话的结构；在这里，
我们把 TA 的生平与说话方式整理成 life.md + persona.md ——
方便你在私人空间里回忆、对稿悼文、或与 AI 对话时「像 TA 还在场」。

开始前请确认：材料来源正当，用途限于私人追思，不用于欺骗或违法。

准备好了吗？
```

---

## 问题序列

### Q1：如何称呼 TA（必填）

```
希望 Skill 里怎么称呼这位逝者？
可以是家人常用称呼、本名、或中性代号（也会用于文件名 slug）。

比如：外公 / 张老师 / 阿明哥
```

**验证**：非空即可。

**Slug 规则**（与 yourself-skill 一致）：

- 中文自动转拼音，用 `-` 连接（「青云」→ `qing-yun`）
- 英文直接小写，空格替换为 `-`（「Little North」→ `little-north`）

---

### Q2：生平概览（可跳过）

```
用一句话概括 TA：年代、职业/身份、常住地、或与你的关系。

可以包括：
- 生卒或年龄段
- 职业 / 社会身份
- 城市
- 教育背景（若重要）

示例：
  「1938 年生，退休中学语文教师，苏州人，我的外公」
  「发小，做过设计师，后来回老家开咖啡店」

跳过也没关系，直接回车。
```

**解析字段**：

- `birth_death_or_era`：年代或生卒概览
- `occupation`：职业/身份
- `city`：城市
- `education`：教育背景（可选）
- `relation_to_builder`：你与 TA 的关系

---

### Q3：性格与印象（可跳过）

```
最后一个：用一句话或几个标签描述 TA？

可以包括：
- 性格特点（可参考下方标签库）
- 邻友或家人常怎么评价 TA
- 口癖或标志性习惯（如爱喝浓茶、常说「算了算了」）

示例：
  「寡言、爱喝茶、面子薄但心软、摩羯座气质但不谈星座」
  「话多、朋友多、开心果型，但生病后话变少了」

跳过也行。
```

**解析字段**：

- `personality`：性格标签列表（可与 PRD 标签库对齐）
- `impression`：主观印象/家属评价

---

## 个性标签库（可选勾选 / 提示用户）

**社交风格**：话痨 · 闷骚 · 社恐 · 社交蝴蝶 · 熟人面前话痨  

**情绪风格**：情绪稳定 · 深夜emo型 · 玻璃心 · 嘴硬心软 · 外冷内热 · 易怒但快消气  

**决策风格**：纠结体 · 果断 · 行动派 · 计划狂 · 凭感觉 · 数据驱动  

**人际模式**：独立 · 粘人 · 边界感强 · 讨好型 · 控制欲 · 没有安全感  

**沟通习惯**：秒回选手 · 已读不回 · 冷暴力 · 讲道理型 · 转移注意力型  

**生活习惯**：早起困难户 · 咖啡依赖 · 极简主义 · 囤积癖 · 报复性熬夜 · 居家派 · 城市漫游者 · 仪式感狂热者  

（与 `docs/PRD.md`、yourself-skill 对齐。）

## 伦理与用途确认（建议置于汇总前）

```
请心里确认（不必逐字打出）：

  □ 材料来源正当，家属或本人（生前）同意范围内使用
  □ 用途为私人追思、家族史或悼念文字整理
  □ 不会用于冒充在世者、诈骗或未经同意的公开曝光

若任一项无法确认，请暂停创建。
```
